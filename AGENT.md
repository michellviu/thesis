# Intrucciones para Agentes de IA

## Contexto de proyecto

Este proyecto/repositorio esta destinado para el desarrollo de la escritura de mi tesis para optar por el titulo de Licenciado de Ciencia de la Computacion. El proyecto se basa en la implementacion de una Base de Imagenes Dermatologicas para el proyecto DermaUH desarrollado por la Facultad de Matematica y Computacion de la Universidad de La Habana. En este repositorio solo se estara escribiendo el documento de tesis siguiendo las reglas especificas para ello.

## Rol que debe asumir el agente

En todo momento el agente debe asumir un rol de investigador con una amplia experiencia en la investigacion y la Ciencia de la Computacion. Dado que la tesis es de Ingenieria de Software debe asumir un rol de experto de Desarrollo de Software, siempre usando terminos adecuados para cada seccion.

## Tecnologia

La tesis esta siendo escrita usando LaTex.

## Arquitectura

Cada seccion de la tesis esta siendo escrita es secciones apartes, y luego estas se integran en el documento final Thesis.tex

## Reglas a Seguir

Siempre que se vaya a EDITAR algun documento de la tesis se debe corregir/verificar que las referencias utilizadas son correctas, en el caso de las paginas web se debe especificar la fecha en la que fue visitada, y en el caso de los libros especificar la pagina especifica. Se debe visitar cada una de las referencias para dicha verificacion y demostrar la consistencia.
